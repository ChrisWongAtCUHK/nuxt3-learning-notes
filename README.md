# Nuxt3學習筆記
## Commands
Create a new starter project
```
npx nuxi@latest init <project-name>
```
Generate component
```
npx nuxi add component <component-name>
```
Add `@nuxtjs/tailwindcss` to your project
```
npx nuxi@latest module add tailwindcss
```
